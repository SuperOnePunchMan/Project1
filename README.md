# Pizza Delivery Api
A RESTFUL API built with **Flask-restful** for managing a pizza ordering system. This api handles customer and admin request such creation of order and  tracking of order.

## Features
-**Role based Access control: for customer and admin
-Order management
-Statu tracking

## Requirements
-Python 3.7+
-Flask
-Flask-Restful
 e.t.c

## Installation and setup
1. **Clone the repository:**
    ```bash
    git clone https://github.com/SuperOnePunchMan/Project1.git

2. **change to project directory:**
    ```bash
    cd "API WITH ME"

3. **create a virtual environment**
    ```bash
    python -m venv env

4. **activate the virtual environment**
    ```bash
    source env/scripts/activate

5. **install dependencies**
    ```bash 
    pip install -r requirements.txt

6. **set flaskapp**
    ```bash
    export FLASK_APP=api

7. **Run the code BoY**
    ```bash
    flask run
